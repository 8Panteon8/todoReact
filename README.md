 # To-do React
 
[hosted](https://todo-5dfb7.web.app)

This is another to-do application but with more functionality.  You can search , delet, complet, allocat your task 

# Features
- Searching
- Deleting
- Completing 
- Highlighting 
- Counter
- Filter

**How to make it work?**

Install the dependencies:
```shell

```
Running a web server:
```shell
npm start 
```
Build app on React:
```shell
npm run build 
```
